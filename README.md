##  Overview
This repository contains end-to-end data engineering pipelines built using PySpark and Delta Lake, following Medallion Architecture (Bronze, Silver, Gold).



# 🏨 Project 1: Hospitality – Dynamic Pricing & Customer Analytics

##  Description
Built a pipeline to analyze hotel bookings and implement dynamic pricing based on demand.

##  Features
- Medallion Architecture
- Dynamic Pricing (Weekend-based multiplier)
- Customer Analytics
- Data Quality Checks
- Partitioning & Z-ORDER Optimization

##  KPIs
- Daily Revenue
- Customer Lifetime Value (CLV)
- Revenue per Hotel



# 🏥 Project 2: Healthcare – Patient Risk & Readmission

##  Description
Developed a pipeline to track patient visits and identify readmissions within 30 days.

##  Features
- Medallion Architecture
- Readmission Detection
- Risk Classification (High/Medium/Low)
- Data Quality Checks
- Optimization

##  KPIs
- Readmission Rate
- Patient Risk Levels
- Total Treatment Cost
